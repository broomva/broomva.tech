# Broomva Commercial Visual — Production Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Produce the three commercial-visual deliverables (16:9 hero still, 1:1 social still, 10s 16:9 video with audio) under the *Anima Toroidalis* brief.

**Architecture:** Two-pass pipeline. (1) Higgsfield generates the visual *substrate* — torus with trajectories on cream paper — via Seedream 4.5 directly (bypassing the gpt_image_2 enhancer that failed in the previous run). (2) Final type is composited in post using hand-authored SVG + `rsvg-convert`, with fonts installed system-wide. The 10s video is generated by Kling 2.6 from the hero substrate as start frame, with type overlay applied via ffmpeg in the final 2 seconds.

**Tech Stack:** Higgsfield CLI (`higgsfield generate create`), Seedream 4.5 (`seedream_v4_5`) for substrate, Nano Banana Pro fallback (`nano_banana_2`), Kling 2.6 (`kling2_6`) AND Cinematic Studio Video 3.0 (`cinematic_studio_3_0`) as parallel video tracks, SVG + `rsvg-convert` (librsvg) for compositing, `ffmpeg` for video overlay and audio post, open-source fonts (Cardo, GFS Didot, STIX Two).

**Spec:** `docs/superpowers/specs/2026-05-11-broomva-commercial-visual-design.md`

**Working directory:** `docs/superpowers/specs/2026-05-11-broomva-commercial-visual-assets/`

**Credit budget:** ~58 credits anticipated (3 substrate iterations × 3c = 9; 3 Kling iterations × 8c = 24; 2 Cinema Studio iterations × 12c = 24 (cinema-grade typically costlier); 1c reserved access check). Starter plan has 85 credits at plan time. Margin: 27c. If Cinema Studio Video 3.0 requires Pro plan (TBD at preflight), that track drops out — budget falls to ~34c.

**Video selection logic:** Both Kling 2.6 AND Cinematic Studio Video 3.0 generate from the same substrate. Final delivery uses the A/B winner based on the brief's acceptance criteria. If Cinema Studio is inaccessible on starter, Kling 2.6 is sole video.

**Acceptance criteria (all must hold):**

If any 3 or fewer fail → proceed to Task 1.2 (iterate).

- [ ] **Step 1:** Identify which acceptance criterion(a) failed for `seedream-01.png` and adjust the prompt.

Common failure modes and their prompt adjustments (apply only the relevant one):

- **Wormy/biological trajectory motion:** Append `, lines drawn with strict geometric precision like CAD output, absolutely no organic deformation or fluid forms, every curve mathematically clean` to the prompt.
- **Missing/weak throat constriction:** Replace "Möbius-like cross-folded throat constriction" with `a clearly visible neck where the torus visibly twists and pinches in a Möbius-like half-twist, distinctly marked in deep sanguine red at the constriction point`.
- **Background too white or too textured:** Replace "warm cream paper background" with `solid warm cream background, hex #FBF5E9, perfectly flat with no texture, no paper grain, no shadows`.
- **Spurious text rendered:** Append `, absolutely no text, no letters, no labels, no captions anywhere in the image — pure illustration only`.
- **Missing marginalia gutters:** Append `, with clearly defined empty rectangular regions on the far left third and far right third of the canvas reserved for marginalia in post-production`.

## Phase 3 — Social 1:1 Compositing

### Task 3.1: Produce the 1:1 social plate

**Files:** `svg/social-plate.svg`, `social/social.png`

Approach: derive from the hero. Two viable methods. Method A (recommended) — author a separate SVG with the 1:1 viewBox that *re-uses* the same substrate cropped center, with type re-flowed to fit a square. Method B — center-crop the rendered hero PNG.

Method A is recommended because the type layout differs between 16:9 and 1:1 (the gutters are too narrow at 1:1 — the table and theorem need to relocate or be omitted). Method B preserves nothing of the typographic discipline at the new aspect ratio.

- [ ] **Step 1:** Center-crop the substrate to 1:1.

```bash
python3 <<'EOF'
from PIL import Image
src = "docs/superpowers/specs/2026-05-11-broomva-commercial-visual-assets/substrate/locked.png"
dst = "docs/superpowers/specs/2026-05-11-broomva-commercial-visual-assets/substrate/locked-square.png"
im = Image.open(src)
w, h = im.size
s = min(w, h)
left = (w - s) // 2
top = (h - s) // 2
im.crop((left, top, left + s, top + s)).save(dst)
print(f"{w}x{h} -> {s}x{s} at {dst}")
EOF
```

Expected: prints the new dimensions. Output PNG exists.
