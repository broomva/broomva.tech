## Running Locally

From the monorepo root:

```sh-session
bun install
bun run dev:landing
```

### Validation commands

```sh-session
bun run typecheck
bun run lint
bun run check:links
bun run test
bun run build
```

### Content locations

- `content/projects/*.mdx`
- `content/writing/*.mdx`
- `content/notes/*.mdx`
- `docs/` for brand, content pillars, templates, and Codex runbooks
