Simple cadence:

- weekly: one substantial post on the website
- daily: one short note that can cross-post to X
- ongoing: capture project deltas directly in project pages

Each note doubles as a distribution surface — it cross-posts, indexes for search, and keeps the site fresh so deeper posts get discovered. Consistency compounds faster than occasional high-effort bursts.
