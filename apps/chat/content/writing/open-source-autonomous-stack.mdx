## Esto no es vibe coding

Es una especie nueva. Construida alrededor de dos ideas fundamentales: **harness engineering** — flujos deterministas con gates de seguridad que hacen que los agentes sean confiables — y **orquestación de agentes** — un daemon que convierte tickets en pull requests autónomos.

Este es el playbook con el que estoy creando sistemas de forma escalable y automática, el mismo flujo con el que he desplegado soluciones que utilizan empresas como FIFA y con el que creo proyectos personales en tiempo récord.

## bstack: 24 Skills, 7 Capas

El stack se llama **bstack** y se instala con un solo comando:

```bash
npx skills add broomva/bstack
```

Las 7 capas cubren todo el ciclo de vida del desarrollo autónomo:

**Foundation** — Control metalayer con safety shields y harness gates. El repositorio se trata como un sistema de control donde los agentes son controladores y el código es la planta.

**Memory** — Consciousness persistente a través de tres sustratos: memoria de trabajo (contexto), knowledge graph (Obsidian vault), y memoria episódica (logs de conversación).

**Orchestration** — [Symphony](https://github.com/broomva/symphony), un daemon Rust que poll-ea trackers de issues y ejecuta sesiones autónomas. [AutoAny](https://github.com/broomva/autoany) implementa EGRI — Evaluator-Governed Recursive Improvement — para mejora recursiva gobernada.

**Research** — Deep research multi-agente con síntesis verificada de 20+ fuentes. Competitive intelligence y catálogo de skills.

**Design** — [Arcan Glass](https://github.com/broomva/arcan-glass), el sistema de diseño web con efectos glass, dark-first, Tailwind v4 + OKLCh.

**Platform** — [Haima](https://github.com/broomva/haima), engine de finanzas agénticas con protocolo x402 para pagos máquina-a-máquina. Content pipeline automatizado.

**Strategy** — 8 skills de decisión: pre-mortem, braindump, morning briefing, drift check, strategy critique, stakeholder updates, decision log, weekly review.

## Life Agent OS: La infraestructura en Rust

Todo corre sobre **Life Agent OS** — un monorepo Rust con 7 crates que implementan un sistema operativo completo para agentes:

- **[Arcan](https://github.com/broomva/arcan)** — Runtime del agente: event loop, lifecycle, multi-provider LLM, SSE streaming
- **[Lago](https://github.com/broomva/lago)** — Persistencia event-sourced: journal append-only, blob store SHA-256, branching FS
- **[Autonomic](https://github.com/broomva/autonomic)** — Controlador homeostático: regulación operacional, cognitiva y económica
- **[Praxis](https://github.com/broomva/praxis)** — Ejecución de herramientas: sandbox, filesystem policy, MCP bridge
- **[Vigil](https://github.com/broomva/vigil)** — Observabilidad: OpenTelemetry nativo, GenAI semantic conventions
- **[Haima](https://github.com/broomva/haima)** — Finanzas: x402, wallet secp256k1, payment policies
- **[Spaces](https://github.com/broomva/lago)** — Networking distribuido: SpacetimeDB, pub/sub inter-agente
